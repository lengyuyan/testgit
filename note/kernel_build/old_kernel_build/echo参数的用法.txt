echo 除了 -n options 之外，常用x有：
        -e ：⒂梅葱本控制字符的DQ(⒖枷卤)
        -E：P]反斜控制字符的DQ(AO如此)
        -n ：取消行末之Q行符(c -e x下的 \c 字符同意)

P於 echo 命令所支援的反斜控制字符如下表：
        \a：ALERT / BELL (南到y喇叭送出)
        \b：BACKSPACE ，也就是向左退格I
        \c：取消行末之Q行符
        \E：ESCAPE，跳I
        \f：FORMFEED，Q字符
        \n：NEWLINE，Q行字符
        \r：RETURN，回I
        \t：TAB，表格跳位I
        \v：VERTICAL TAB，垂直表格跳位I
        \n：ASCII 八M位a(以 x _首槭六M位)
        \\：反斜本身
